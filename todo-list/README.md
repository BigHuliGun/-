## To-Do List Application

Простое приложение для управления списком задач с сохранением в браузер.

### Функциональность

✅ **Добавление задач** - введи текст и нажми "Добавить" или Enter
✅ **Отметить как выполненную** - галочка рядом с задачей
✅ **Удалить задачу** - кнопка "×"
✅ **Фильтры** - Все, Активные, Выполненные
✅ **Счётчик задач** - показывает активные и всего задач
✅ **Очистить выполненные** - удалить все завершённые задачи
✅ **Локальное хранилище** - все задачи сохраняются в браузер

### Как использовать

1. Открой `index.html` в браузере
2. Введи текст задачи в поле ввода
3. Нажми "Добавить" или Enter
4. Щёлкни галочку, чтобы отметить как выполненную
5. Нажми "×", чтобы удалить задачу
6. Используй фильтры для просмотра разных категорий

### Сохранение данных

Все задачи автоматически сохраняются в `localStorage` браузера. Они будут доступны даже после закрытия и перезагрузки страницы.

- Чёрно-белый минимализм
- Адаптивный дизайн для мобильных устройств

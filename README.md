# Квитанции

Веб-приложение для оформления квитанций. Лаконичный чёрно-белый дизайн.

## Функциональность

- ✅ Номер квитанции (кв. №)
- ✅ Поле "Заказчик" (Имя Фамилия)
- ✅ Поле "Телефон" с иконками мессенджеров:
  - Telegram
  - WhatsApp
  - Messenger
- ✅ Дата приема
- ✅ Дата выдачи
- ✅ Печать документа
- ✅ Очистка формы

## Использование

1. Откройте `index.html` в браузере
2. Заполните поля квитанции
3. Нажмите "Печать" для вывода на печать или Ctrl+P
4. Нажмите "Очистить" для сброса формы

